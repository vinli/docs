Virtual Vinli Dummy Service
---------------------------

Create a Dummy Device
`````````````````````

Request
+++++++

::
	
  POST https://dummies.vin.li/api/v1/dummies
  Content-Type: application/json

  {
     "dummy":{
        "name":"YourDummyName"
     }
  }
